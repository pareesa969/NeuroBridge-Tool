# 🧠 LUMINESCENCE AI

---

# Project Title

**LUMINESCENCE AI — Adaptive Learning & Emotional Support Platform**

An AI-powered productivity and learning assistant that adapts to user focus, reading ability, and emotional state.

---

# Problem Statement

Students and professionals struggle with:

* Lack of focus during study sessions
* No personalized learning support
* Poor reading comprehension tools
* No centralized productivity dashboard
* No emotional / AI learning assistant
* Scattered study organization methods

Most existing tools solve **only one problem** — not the complete learning workflow.

Users need **one unified AI-powered system** for learning, focus, and productivity.

---

# Solution

LUMINESCENCE AI provides an **adaptive learning environment** powered by AI that:

* Analyzes user learning behavior
* Provides focus tools
* Offers reading support
* Organizes study workflow
* Tracks progress
* Uses Gemini AI for smart assistance

The platform creates a **personalized learning sanctuary** for users.

# How to Run

## 1. Clone Repository

```bash
git clone https://github.com/pareesa969/Luminescence-AI.git
```

## 2. Navigate to Folder

```bash
cd luminescence-ai
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Setup Environment Variables

Create `.env`

```
VITE_GEMINI_API_KEY=your_api_key_here
```

Or use example file

```
cp .env.example .env
```

---

## 5. Run Development Server

```bash
npm run dev
```

---

## 6. Build for Production

```bash
npm run build
```
